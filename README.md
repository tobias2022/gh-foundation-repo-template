1. Rename the repository and set a clear description/topics.
2. Replace all placeholder values in this repository.
3. Update `.github/CODEOWNERS` with real teams and owners.
4. Review `CONTRIBUTING.md`, `SECURITY.md`, and `SUPPORT.md`.
5. Confirm repository visibility and team permissions per organizational policy.
6. Confirm branch protection/rulesets for the default branch.
7. Confirm required checks and code-owner approval requirements are enabled.

- Code owner reviews required
- Required status checks must pass
- Force pushes disabled
- Branch deletion disabled
